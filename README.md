Repository for class project Spoil Less
___________________________________________

Spoil-Less is an application that helps people use their groceries before they spoil by making expiration tracking quick and easy. The app simplifies food tracking, allowing users to easily add things they want to track, see what they have, what is expiring soon, and what should be used as soon as possible.
